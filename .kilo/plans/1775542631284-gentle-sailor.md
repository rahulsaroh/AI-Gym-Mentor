## KNOWN LIMITATIONS

1. **Firebase** - Disabled in main.dart (commented out) - needs google-services.json
2. **Google Sheets Sync** - Requires Google Cloud setup as documented in main.dart
3. **Background Services** - Disabled for debug (commented out in main.dart)
4. **Notifications** - Will fail without proper Android configuration

These are configuration-dependent and not code bugs.
